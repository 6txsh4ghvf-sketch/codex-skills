# 来源规则

## 来源等级

| 等级 | 来源类型 | 使用方式 |
| --- | --- | --- |
| A级 | 官方机构、国际组织、法定公告、权威统计发布 | 可作为专报主要支撑 |
| B级 | 行业协会、专业研究机构、主流数据库、头部咨询机构 | 可支撑产业趋势、市场判断、价格周期、物流风险 |
| C级 | 主流财经媒体、专业媒体、企业公告 | 可作为线索或背景，涉及政策、数据和机构判断时追溯原始来源 |
| D级 | 普通转载、自媒体、论坛、无法核实来源内容 | 不得直接作为专报判断依据 |

## 使用规则

1. A级、B级来源优先。
2. C级来源谨慎使用，不能替代政策、法规、数据的原始来源。
3. D级来源最多作为线索，必须进一步核验。
4. 来源清单是优先参考，不是封闭白名单。
5. 新增来源必须记录机构属性、关注领域、适用场景、可信度判断和使用边界。
6. 对口径不明、来源不清、发布时间不明的信息，标注“待核实”。

## 媒体使用边界

媒体报道可用于发现线索、补充背景和观察市场情绪。涉及政策、数据、机构判断、法规文本、贸易救济、市场准入、技术标准时，应尽量追溯官方机构、国际组织、行业协会、专业研究机构或原始公告。

无原始出处的转载文章、自媒体和论坛内容不得直接进入专报判断。

## 联网核验规则

仅当用户明确要求“检索”“核验”“监测”“查找最新信息”“搜索公开来源”“联网确认”等任务时，使用 Browser、Chrome 或联网能力访问公开网站。

联网时：

1. 只能访问公开网站。
2. 不得访问单位内部系统、非公开平台、个人邮箱、网盘、涉密资料或敏感资料。
3. 每条联网信息记录来源、发布时间、链接或出处说明、来源等级、核验结论和待核实事项。
4. 无法确认原始来源时，标注“待核实”或“仅作线索参考”。
5. 不为凑数量使用低质量来源。
