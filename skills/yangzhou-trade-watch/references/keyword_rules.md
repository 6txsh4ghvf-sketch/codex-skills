# 关键词规则

## 组合方法

检索和判断优先使用“市场 + 产业 + 规则 + 扬州关注对象”的组合方式。

示例：

- 美国 + 关税 + 机械设备 + 中国出口
- 欧盟 + CBAM + 化工品 + 碳规则
- 墨西哥 + USMCA + 机床 + 近岸化
- 中东 + 红海局势 + 海运费 + 机械设备
- WTO + 全球贸易展望 + 货物贸易
- Drewry + 集装箱运价 + 亚欧航线
- 光伏 + 组件价格 + 海外需求 + 贸易救济
- 船舶 + 全球新船订单 + 绿色船舶

## 重点市场

美国、欧盟、东盟、日本韩国、墨西哥、中东、非洲、印度、拉美、英国、澳大利亚等。

## 重点产业

船舶、光伏、平板显示模组、电子信息、机械设备、农机、汽车零部件、交通装备、化工品、医药材、农食产品。

## 重点规则和风险

关税、贸易救济、出口管制、TBT、SPS、CBAM、REACH、RoHS、USMCA、RCEP、原产地规则、红海局势、霍尔木兹海峡、油价、海运费、战争险、港口拥堵。

## 扬州关注对象

船舶、太阳能电池、平板显示模组、基本有机化学品、农业机械、医药材、机械设备、汽车零部件、玩具、塑料制品、机床、进口平板显示模组、进口催化剂、进口电子纸母片、资源类产品、农食产品。

## 判断规则

1. 检索结果不能直接写入专报。
2. 先判断来源等级，再摘录事实。
3. 必须说明是否关联扬州重点产品、重点市场、企业类型或本地数据口径。
4. 无扬州关联的国际动态，原则上不作为专报重点。
5. 没有本地数据支撑的内容，标注“需匹配本地数据后再使用”。
