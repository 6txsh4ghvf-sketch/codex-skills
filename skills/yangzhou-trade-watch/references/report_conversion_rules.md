# 专报段落转化规则

## 核心顺序

专报段落不能写成新闻综述，必须按以下逻辑组织：

1. 先给总体判断；
2. 后用权威来源支撑；
3. 再落到扬州数据、产品、市场或企业；
4. 再说明可能影响；
5. 最后提出面向市政府的现实可行建议。

## 输出结构

1. 小标题；
2. 一句话判断；
3. 权威来源支撑；
4. 扬州数据或产业关联；
5. 企业或产品影响；
6. 面向市政府的建议落点；
7. 可直接修改使用的专报段落草稿。

## 写作要求

1. 不写成新闻综述。
2. 不堆砌国际动态。
3. 不写空泛判断。
4. 不夸大影响。
5. 不把建议写成海关自身工作安排。
6. 语言克制、直白、清晰。
7. 尽量形成“问题-影响-建议”的闭环。
8. 没有本地数据支撑的内容，标注“需匹配本地数据后再使用”。
9. 建议必须面向市政府层面，现实可行。

## 段落压缩规则

1. 信息卡片可以详细，但专报段落必须压缩。
2. 专报段落不宜罗列过多产品和市场，只保留最相关的1-3个对象。
3. 每段原则上突出1个主判断、1组关键影响、1个建议方向。
4. 对缺少本地数据支撑的内容，可在段末统一标注“需匹配本地数据后再使用”，不要在段落中反复打断表述。
5. 如果一条动态同时涉及多个市场、多个产业或多个风险，应优先写与扬州关联最强、最可转化为建议的部分。

## 影响分层规则

月度观察和专报转化时，应优先写主要影响，避免把所有可能影响平铺。

| 层级 | 含义 | 写法 |
| --- | --- | --- |
| 主要影响 | 与扬州重点产品、重点市场或企业订单最直接相关 | 优先进入一句话判断和正文 |
| 次要影响 | 有一定关联，但传导链条较长或影响较弱 | 可简写或放入背景 |
| 结构性机会 | 可能带来市场拓展、进口增量、产业升级或新需求 | 谨慎表达为“可关注”“可结合数据观察” |
| 中长期风险 | 规则、成本、产业周期、供应链重构等长期压力 | 适合写入持续跟踪或建议部分 |
| 待观察事项 | 来源、口径、适用范围或本地传导尚不明确 | 标注“待核实”或“需持续跟踪” |

## 绿色规则和技术规则表述边界

涉及欧盟绿色规则、CBAM、REACH、RoHS、电池法规、FDA、FSVP、TBT、SPS 等内容时，不得笼统认定所有相关产品均直接适用。

1. 直接适用范围需核验官方规则、商品范围、产品编码、执行阶段和口径。
2. 间接受影响可从客户合规要求、供应链尽责、碳数据、认证成本、订单维护和市场准入预期等角度分析。
3. 正式写入专报前必须核验官方适用范围和产品口径。
4. 如仅能判断存在间接影响，应写成“可能通过客户合规要求、认证成本等渠道传导”，不要写成“已直接受限”。

## 建议落点表达

建议应面向市政府层面，优先使用有对象、有抓手、有动作的表达：

1. 分行业梳理重点企业清单。
2. 开展规则适用范围和订单影响摸排。
3. 组织政策解读和第三方认证对接。
4. 支持企业拓展多元市场。
5. 关注物流成本、订单交付和企业利润变化。
6. 引导企业提前完善碳数据、合规材料和客户沟通机制。

避免空泛使用“加强监测”“强化服务”“完善机制”等表述；如确需使用，后面必须写明对象、抓手和具体动作。

## 不建议转化的情况

1. 来源无法核实。
2. 无法落到江苏、扬州外贸。
3. 无重点产品、重点市场或企业类型关联。
4. 只能形成泛泛观点，不能形成现实建议。
5. 存在明显表述风险或事实不确定。
