# 专报段落转化模板

> 专报段落应遵循“总体判断-权威来源-扬州关联-企业影响-建议落点”的顺序，不要从新闻本身开始平铺。
> 信息卡片可以详细，但专报段落必须压缩。每段原则上突出1个主判断、1组关键影响、1个建议方向。

## 1. 标题

标题要直给问题或判断，避免空泛。

待填写。

## 2. 一句话判断

先说明这条外部动态对扬州外贸意味着什么。

待填写。

## 3. 权威来源支撑

| 字段 | 内容 |
| --- | --- |
| 来源机构 | 待填写 |
| 发布时间 | 待填写 |
| 来源等级 | A/B/C/D |
| 核心依据 | 待填写 |
| 是否已核验 | 是/否/待核实 |

## 4. 扬州数据或产业关联

| 关联项 | 内容 |
| --- | --- |
| 关联产品 | 待填写；原则上保留最相关的1-3个 |
| 关联市场 | 待填写；原则上保留最相关的1-2个 |
| 关联企业类型 | 待填写 |
| 需要使用的本地数据 | 待填写 |
| 数据说明的问题 | 待填写 |
| 适用关系 | 直接适用/间接受影响/待核实/仅作背景 |

## 5. 企业或产品影响

说明可能影响订单、价格、物流、合规、成本、进口来源或市场开拓。优先写主要影响，其他影响按需压缩。

| 影响层级 | 内容 |
| --- | --- |
| 主要影响 | 待填写 |
| 次要影响 | 待填写 |
| 结构性机会 | 待填写 |
| 中长期风险 | 待填写 |
| 待观察事项 | 待填写 |

## 6. 建议落点

建议必须面向市政府层面，现实可行，不写成海关自我工作安排。优先使用“分行业梳理重点企业清单、开展规则适用范围和订单影响摸排、组织政策解读和第三方认证对接、支持企业拓展多元市场、关注物流成本和订单交付、引导企业完善碳数据和客户沟通机制”等有抓手的表达。

待填写。

## 7. 可直接写入专报的段落

形成一段成熟、克制、适合市政府阅读的文字。应体现“数据说明什么、风险在哪里、建议落在哪里”。缺少本地数据支撑时，可在段末统一标注“需匹配本地数据后再使用”，不要在段内反复出现。

待填写。

## 8. 规则适用提示

涉及 CBAM、REACH、RoHS、电池法规、FDA、FSVP、TBT、SPS 等规则时，必须区分直接适用、间接受影响、待核实或仅作背景。不得笼统认定所有相关产品均直接适用。
