# 国际经贸动态信息卡模板

> 一张卡片只记录一个主题。事实摘录和影响判断必须分开；无法核实的信息统一标注“待核实”。

## 一、基础信息

| 字段 | 内容 |
| --- | --- |
| 动态标题 | 待填写 |
| 信息来源 | 待填写 |
| 来源链接或出处说明 | 待填写 |
| 发布时间 | 待填写 |
| 信息获取时间 | 待填写 |
| 来源等级 | A/B/C/D，待判断 |
| 涉及国家或地区 | 待填写 |
| 涉及产业 | 待填写 |
| 涉及政策、规则或风险 | 待填写 |
| 对应关键词 | 待填写 |
| 信息卡片编号 | 待填写 |

## 二、事实摘录

| 字段 | 内容 |
| --- | --- |
| 原文关键信息摘录 | 待填写 |
| 核心事实摘要 | 待填写 |
| 涉及数据 | 待填写；如无数据，写“无明确数据” |
| 涉及机构判断 | 待填写；如无机构判断，写“无明确机构判断” |
| 涉及企业或行业案例 | 待填写；如无案例，写“无明确案例” |
| 是否为官方发布 | 是/否/待核实 |
| 是否为媒体报道 | 是/否/待核实 |
| 是否需要进一步追溯原始来源 | 是/否；说明原因 |

## 三、来源核验

| 核验项 | 结论 |
| --- | --- |
| 来源等级 | A/B/C/D，待判断 |
| 是否有原始出处 | 是/否/待核实 |
| 是否可公开检索 | 是/否/待核实 |
| 是否有多个来源交叉印证 | 是/否/暂未交叉验证 |
| 是否存在口径不明 | 是/否/待核实 |
| 是否存在时间滞后 | 是/否/待核实 |
| 是否存在商业推广、观点倾向或二手转述 | 是/否/待核实 |
| 核验结论 | 可采用/可作背景/需继续核实/暂不采用 |

## 四、扬州关联判断

| 字段 | 内容 |
| --- | --- |
| 对全球贸易形势的影响 | 待填写 |
| 对中国外贸的可能影响 | 待填写 |
| 对江苏外贸的可能影响 | 待填写 |
| 对扬州外贸的可能影响 | 待填写 |
| 关联扬州重点产品 | 待填写 |
| 关联扬州重点市场 | 待填写 |
| 关联扬州重点企业类型 | 待填写 |
| 适用关系 | 直接适用/间接受影响/待核实/仅作背景 |
| 可能影响方向 | 需求增加/需求减弱/成本上升/物流受阻/合规压力增加/订单转移/市场替代/价格下行/进口机会/出口风险/政策窗口/待判断 |
| 影响强度 | 高/中/低/待观察 |
| 影响时点 | 当月可能影响/未来1-3个月可能影响/上半年可能影响/全年可能影响/中长期影响/暂不明确 |
| 影响分层 | 主要影响/次要影响/结构性机会/中长期风险/待观察事项 |
| 是否适合写入月度专报 | 是/否/待观察 |
| 适合写入专报的位置 | 总体判断/重点市场动态/重点产业动态/贸易规则和物流风险/对扬州外贸影响/建议持续跟踪事项/暂不适用 |
