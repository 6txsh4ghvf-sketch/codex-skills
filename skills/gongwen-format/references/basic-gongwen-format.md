# 公文基本格式基线

Use this reference when formatting a Word document as 公文基本格式.

## 一、公文标题

### 结构

发文机关 + 事由 + 文种。

### 格式

- 字体字号：方正小标宋_GBK，二号。
- 对齐：居中。
- 行距：固定值 30 磅。

### 注意事项

- 可以使用书名号、逗号、顿号、连接号、括号等标点，但不应滥用。
- 标题排列应尽量呈梯形或菱形，避免长方形、沙漏型、金字塔型。
- 回行应词意完整、排列对称、长短适宜、间距恰当。

## 二、正文

### 间距和页面

- 行间距：固定值 28 磅。
- 字符间距：压缩 0.2 磅。
- 页边距：上 3.8 cm，下 3.5 cm，左 2.7 cm，右 2.7 cm。
- 对齐方式：两端对齐。

### 字体

- 一级标题：方正黑体_GBK，三号。例：`一、基本情况`。
- 二级标题：方正楷体_GBK，三号；独立成行时标题后加句号。例：`（一）基本情况。`
- 二级标题如为一段话，可用方正仿宋_GBK，三号。
- 三级标题：方正仿宋_GBK，三号，加粗；序号后加一个空格；独立成行时标题后加句号。例：`1. 基本情况。`
- 四级标题：方正仿宋_GBK，三号；独立成行时标题后加句号。例：`（1）基本情况。`
- 正文内容：方正仿宋_GBK，三号。
- 阿拉伯数字和英文：Times New Roman，三号。

### 结束语

- 请示：`特此请示`、`妥否，请示复`。
- 通知：`特此通知`。
- 报告：`特此报告`。
- 函：`特此致函`、`特此函报`、`特此函复`。
- 通报：`特此通报`。
- 批复：`此复`、`特此批复`。

### 落款

一般为 `行文单位 + 日期`，置于正文下空 3 行。

### 附注

请示、重要报告、需要沟通协调的通知、涉及请求答复事项的函等，应附注联系人和联系电话。例：`联系人：……，联系电话：……`。

### 页码

- 公文版记页前有空白页的，空白页和版记页均不编排页码。
- 附件与正文一起装订时，页码应连续编排。
- 奇偶页不同：奇数页页码在页面右下角，偶数页页码在页面左下角。
- 字体：宋体四号，半角格式。例：`— 1 —`。

### 其他注意事项

- 引用文件需在文件名后添加文号。例：`《XX文件》（扬关办发X号）`。
- 文号年份使用六角括号 `〔〕`，不得用 `【】` 替代。
- 单书名号 `〈〉` 不得用大于号、小于号代替。
- 连用的书名号、引号之间一般不加顿号。
- 公文中的标点符号应为全角。例：`，`、`：`，不是 `,`、`:`。

## 三、附件

附件是公文正文的说明、补充或者参考资料。

- `附件` 二字及附件序号：方正黑体_GBK，三号，编排在版心左上角第一行。
- 附件标题：居中编排在版心第三行。
- 附件序号和标题应当与附件说明的表述一致。
- 附件正文格式要求同正文。
